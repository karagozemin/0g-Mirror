# 0G Mirror Demo Flow

Use this as a spoken 90-second judge demo.

“AI agents are becoming decision-makers, but their decisions are still black boxes.”

## 0–10s — Problem

Open on the landing page and state the problem: AI agents already act in protocols, but their decision evidence is hidden. Explain that auditors and users cannot reproduce or attest those outcomes today.

## 10–35s — Mirror Core

Navigate to Mirror Core. Pick an agent and a task, then generate a Decision Trace. Show these items on screen: the evidence list, the public rationale, and the decision hash. Explain briefly that the Decision Trace packages inputs, evidence, model metadata, tools used, and a public rationale into a versioned JSON.

## 35–60s — Real 0G Proof

Upload the trace to 0G Storage and register it on-chain. On-screen, point to the 0G Storage URI and the transaction links. Show the decision hash and confirm the on-chain registration. Run replay verification and call out the result (Verified / Inconsistent / Missing Evidence). Emphasize: the proof is the combination of the stored JSON, the storage root/URI, and the on-chain attestation.

## 60–85s — Olympus Arena

Open Olympus Arena and stress this is a showcase mode built on top of Mirror Core (not a separate project). Run two deterministic agents on the same prompt, show both Decision Traces side-by-side (evidence lists and public rationales), then run the Olympus Judge. Show the Court Verdict Card with winner, reason codes, verdict root, and the attestation.

## 85–90s — Close

Close with a concise takeaway: “0G Mirror is not another AI agent app. It is the verification layer that makes agent decisions auditable on 0G.”

During the demo, explicitly point to these moments:

- Show evidence list
- Show public rationale
- Show decision hash
- Show 0G Storage URI
- Show on-chain registration/status
- Show Verified / Inconsistent / Missing Evidence
- Show Olympus Court Verdict Card
