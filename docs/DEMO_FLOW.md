# 0G Mirror Demo Flow

Use this as a spoken 90-second demo script.

1. Open the landing page and frame the problem: AI agents are already making decisions, but the evidence trail is usually hidden.
2. Say that 0G Mirror turns each decision into a public Decision Trace that can be stored, replayed, and attested.
3. Open Mirror Core.
4. Pick an agent and task, then generate a Decision Trace.
5. Show the evidence list, the public rationale, and the decision hash.
6. Store the trace on 0G, then register it on 0G Chain so the proof is visible on-chain.
7. Run replay verification and show the result change to Verified, Inconsistent, or Missing Evidence.
8. Open Olympus Arena and call out that this is the showcase mode, not a separate project.
9. Run the two-agent challenge and show both traces side by side.
10. Appeal to Olympus and explain that the judge compares evidence coverage and replay status.
11. Show the Court Verdict Card with the winner, reason codes, verdict root, and attestation.
12. Close with: “0G Mirror is not another AI agent app. It is the verification layer that makes agent decisions auditable on 0G.”
